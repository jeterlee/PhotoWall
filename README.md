## PhotoWall
**项目说明：Android照片墙，使用了三级缓存，瀑布流式显示，多点触控缩放和移动照片，使用ViewPager实现画廊。**


实现技术：
- 1. 使用 LruCache、DiskLruCache 实现缓存。
- 2. 使用瀑布流式显示照片。
- 3. 多点触控缩放和移动照片。
- 4. 使用ViewPager实现画廊效果。
